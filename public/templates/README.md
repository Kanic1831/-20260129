# 周计划模板说明

请将您的周计划 Word 模板文件放在此目录下。

## 模板文件命名

请将模板文件命名为：`weekly_plan_template.docx`

## 占位符格式

模板中的占位符应使用双花括号 `{{ }}` 格式，例如：
- `{{上周回顾}}`
- `{{周主题}}`
- `{{活动目标}}`
- `{{周一上午活动}}`
- `{{周一下午活动}}`
- `{{周二上午活动}}`
- `{{周二下午活动}}`
- `{{周三上午活动}}`
- `{{周三下午活动}}`
- `{{周四上午活动}}`
- `{{周四下午活动}}`
- `{{周五上午活动}}`
- `{{周五下午活动}}`
- `{{环境创设}}`
- `{{家园共育}}`
- `{{注意事项}}`
- `{{活动反思}}`
- `{{思考与调整}}`

## 使用方式

- 如果您的模板文件名为 `weekly_plan_template.docx`，系统会自动使用它
- 您也可以选择上传自定义模板来覆盖默认模板
- 如果没有默认模板，则需要上传模板文件
